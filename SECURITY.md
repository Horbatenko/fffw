# Security Policy

## Supported Versions

Currently only 2.0.x branch has full support, 1.0.x branch is shut down.

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.0.x   | :x:                |


## Reporting a Vulnerability

Please, open an issue at github.

We'll try to respond in two weeks.
